# Photopea Automation Assignment

This project automates basic design operations on Photopea using Selenium WebDriver and Java.

## Tools & Technologies

- Java
- Selenium WebDriver
- Maven
- Spring Tool Suite
- Git

## Website Automated

https://www.photopea.com/

## Automation Features

The following tasks are automated:

1. Launch Photopea website
2. Create a new canvas
3. Set canvas size:
   - Width: 1280
   - Height: 960
4. Add text "Hello"
5. Add text "World"
6. Draw shapes:
   - Rectangle
   - Circle
7. Export design as PNG
